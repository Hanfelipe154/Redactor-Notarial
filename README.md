Redactor Notarial 

Herramienta desarrollada en Python para la automatización y agilización de generacion de poderes para firmas de escrituras. 

¿Que problema que resuelve?
En el sector notarial, la redacción manual de documentos es propensa a errores y consume mucho tiempo. Este proyecto estandariza la entrada de datos 
y genera documentos precisos de forma automática, sin dejar aun lado el estilo y la organizacion del mismo.

¿Que Tecnologías usa?
- Lenguaje: Python 3.x
- Interfaz de Usuario: Streamlit, para crear una aplicación web interactiva y amigable.
- Procesamiento de Documentos: python-docx, para la manipulación y generación automática de archivos Word.
- Entorno y Despliegue: Git para control de versiones, PyInstaller para empaquetar la aplicación y Visual Studio Code como IDE.

¿Cuales son sus funcionalidad?
- Generación de plantillas dinámicas.
- Validación de datos de entrada.
- Exportación a formato Docx.

Para realizar la Instalación:

1. Clonar el repositorio: `git clone https://github.com/Hanfelipe154/Redactor-Notarial.git`
2. Crear entorno virtual: `python -m venv venv`
3. Activar entorno e instalar dependencias: `pip install -r requirements.txt`
